# AcePrep AI 🚀

An AI-powered Interview Preparation Platform that helps users generate personalized interview strategies, technical interview questions, behavioral interview questions, skill gap analysis, preparation roadmaps, and ATS-style match scores based on a job description and the user’s profile/resume.

🌐 **Live Demo:**  
https://ace-prep-ai-sage.vercel.app

---

# 📌 About The Project

AcePrep AI was built to solve a very common problem faced by students, freshers, and job seekers:

> “How do I prepare specifically for this job role?”

Most candidates prepare randomly without knowing:
- whether their profile matches the role,
- what technical questions may be asked,
- what skills are missing,
- or how strong their profile is for that position.

AcePrep AI solves this by using AI to analyze:
- the job description,
- the candidate’s resume,
- and their self-description,

then generates a complete personalized interview preparation strategy.

# ✨ Features

## 🔐 Authentication System
- User Registration & Login
- JWT-based Authentication
- Persistent Login
- Mobile + Desktop compatible auth flow

---

## 🤖 AI-Powered Interview Report Generation

Users can:
- Paste a Job Description
- Upload Resume (PDF)
- Or write a Self Description

The AI then generates:
- Technical Interview Questions
- Behavioral Interview Questions
- ATS-style Match Score
- Skill Gap Analysis
- Personalized Preparation Plan

---

## 📄 Resume PDF Support
- Resume Upload using Multer
- PDF Parsing using pdf-parse

---

## 📊 ATS-Style Match Score

The platform compares:
- User skills
- Experience
- Resume content
- Job requirements

and generates a compatibility score based on the target role.

---

## 📱 Fully Responsive
Works across:
- Desktop
- Android
- iPhone

---

## 📂 Interview History
Users can revisit all previously generated interview reports anytime.
